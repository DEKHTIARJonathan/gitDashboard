Manual:
  • Permission
    • R: for read only
    • RW: for push existing ref or create new ref
    • RW+:for "push -f" or ref deletion allowed
    • - (the minus sign): to deny access
    • Advanced Permission
  • User Group
    • Simple association
      @group1 = user1 user2
      same result:
      @group1 = user1
      @group1 = user2

    • Aggregate group
      @group1= user1 user2
      @group2= user3 user4
      @developer = @group1 @group2 user5
    • Group Doc
  • Example
    repo test.git
        RW+    =    user1
        R      =    @group1
    repo /subdir/subRepos.test
        RW+    =    user1
        R      =    @group1
